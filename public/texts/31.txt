Задача 31
1.	Спроектируйте базу данных University для автоматизации учета информации о филиалах учебного заведения.
Учебное заведение имеет несколько филиалов. Для каждого филиала должна храниться следующая информация: 
•	номер филиала (уникален, содержит только цифры);
•	адрес;
•	дата последнего подтверждения Государственной аккредитации. 
В учебном заведении хранится информация обо всех группах каждого из филиалов:
•	номер филиала, к которому относится группа;
•	Номер группы (имеет уникальное значение);
•	курс;
•	количество учащихся в группе.  
В СУБД  MS SQL SERVER,  используя команды определения данных создать базу данных.  Создать таблицы Филиал и Группа, для каждой таблицы определив первичные ключи. Определить в подчиненной таблице внешний ключ. Создать связь между таблицами с помощью внешнего ключа, обеспечив каскадное удаление и обновление данных. (+1)
2.	Используя команды манипулирования данными, добавить в созданные таблицы по пять записей, таких чтобы результаты нижеперечисленных запросов содержали не менее одной записи. (+1)
  
3.	Написать  запросы для получения следующей информации (результаты выполнения запросов не должны быть пустыми):
1)	Выберите номера групп и номера филиалов, в которых число учащихся ниже среднего. (+1)
2)	Выведите адреса и номера всех филиалов, которые не подтверждали Государственную аккредитацию в текущем году. (+1)
3)	Выведите номер филиала, адрес филиала и номера групп в которых количество учащихся меньше 25. (+1)
4)	Выведите для каждого филиала его номер и количество групп четвертого курса. (+1)


-- 1. Создание БД и таблиц
CREATE DATABASE University;
GO
USE University;
GO

CREATE TABLE Филиал (
    номер INT PRIMARY KEY CHECK (номер LIKE '[0-9]%'),
    адрес VARCHAR(255),
    дата_аккредитации DATE
);
GO

CREATE TABLE Группа (
    номер_группы INT PRIMARY KEY,
    номер_филиала INT,
    курс INT,
    количество_учащихся INT,
    FOREIGN KEY (номер_филиала) REFERENCES Филиал(номер)
    ON DELETE CASCADE ON UPDATE CASCADE
);
GO

-- 3. SQL-запросы

-- 1) Группы с числом учащихся ниже среднего
SELECT номер_группы, номер_филиала
FROM Группа
WHERE количество_учащихся < (SELECT AVG(количество_учащихся) FROM Группа);

-- 2) Филиалы без аккредитации в текущем году
SELECT номер, адрес
FROM Филиал
WHERE YEAR(дата_аккредитации) != YEAR(GETDATE());

-- 3) Филиалы и группы с количеством учащихся меньше 25
SELECT f.номер, f.адрес, g.номер_группы
FROM Филиал f
JOIN Группа g ON f.номер = g.номер_филиала
WHERE g.количество_учащихся < 25;

-- 4) Количество групп 4 курса для каждого филиала
SELECT f.номер, COUNT(g.номер_группы) AS количество_групп_4_курса
FROM Филиал f
LEFT JOIN Группа g ON f.номер = g.номер_филиала AND g.курс = 4
GROUP BY f.номер;
